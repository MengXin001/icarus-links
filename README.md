## 配置
创建`Hexo根目录\source\links\`
在links文件夹中导入`index.md`模板
在`Hexo根目录\themes\icaurs\layout`中导入`links.ejs`
links.ejs中可以修改顶部的简介
index.md文件参照样式修改即可，不建议使用Notepad++编写导致出现缩进问题。
